# Google Antigravity - ユースケース

## 概要
Antigravityはエージェントファーストのプラットフォームです。エージェントに指示が与えられると、エージェントは単独でタスクを実行し、必要に応じて権限を要求し、アーティファクトを生成し、タスクが完了すると通知します。

## 主なユースケース

### 1. ニュースハイライト
**目的：** ウェブブラウザを使用してウェブサイトにアクセスし、情報を抽出し、アクションを実行してユーザーにデータを返す

**実行内容：**
- Googleニュースのサイトにアクセスして情報を抽出
- エージェントがブラウザを起動し、サイトに自動アクセス
- サイトを移動して情報を取得
- アーティファクトとして結果を生成

**特徴：**
- ブラウザの周りの青い枠線は、エージェントがブラウザを制御していることを示す
- 思考プロセスが左側に表示される

**応用例：**
- 任意のウェブサイトを選択し、エージェントにアクセスしてデータを取得または要約してもらう
- ダッシュボードとグラフがあるウェブサイトから値を選択するように依頼
- リリースノートの要約をカテゴリ別に取得

### 2. Python + Flaskを使用した動的ウェブサイト生成
**目的：** 完全なウェブアプリケーションを生成

**プロジェクト例：** 1日限りの技術イベント情報サイト

**要件：**
- ホームページに現在の日付、場所、スケジュール、タイムテーブルを表示
- 1日のイベントは合計8つの講演
- 各講演には最大2人のスピーカー
- 講演にはID、タイトル、スピーカー、カテゴリ、説明、時間を含む
- 各スピーカーには名前、姓、LinkedInのURLを含む
- カテゴリ、スピーカー、タイトルで検索可能
- 60分の昼休憩を設定
- ダミーデータを使用（Google Cloud Technologiesに関するイベント）

**技術スタック：**
- サーバーサイド：Python、Flaskフレームワーク
- フロントエンド：基本的なHTML、CSS、JavaScript

**エージェントの動作：**
- タスクアーティファクトの作成
- 実装アーティファクトの作成
- チュートリアルアーティファクトの作成
- すべての機能をテスト
- 詳細なREADMEを提供（セットアップ、実行、変更方法）
- ウェブアプリケーションを起動してレビュー可能にする

### 3. その他の活用方法
- 外部サイトでのタスク自動化
- プロジェクトの単体テストケースの生成と検証
- ウェブサイトの完全な開発

## 主要な機能

### エージェントマネージャー
- ミッションコントロールダッシュボードとして機能
- 高レベルのオーケストレーション
- Playgroundでの会話開始

### ブラウザ制御
- エージェントがChromeブラウザを自動制御
- ウェブサイトへの自動アクセスと情報抽出
- 青い枠線でエージェントの制御を視覚化

### アーティファクト生成
- タスクリスト
- 実装計画
- ウォークスルー
- スクリーンショット
- チュートリアル
