## 「Microsoft 365 Copilot」に強力なエージェント「Researcher」と「Analyst」が追加

専門家のように徹底的な調査や「Python」を用いた大量・複雑な業務データ分析が可能に

[![](https://asset.watch.impress.co.jp/img/wf/docs/2002/484/image_top_l.png)](https://forest.watch.impress.co.jp/img/wf/docs/2002/484/html/image_top.png.html) 「Microsoft 365 Copilot」に頼りになるエージェント「Researcher」と「Analyst」が追加

　米Microsoftは3月25日（現地時間）、業務用の新しい推論エージェント「Researcher」「Analyst」を「Microsoft 365 Copilot」に追加したと発表した。メールやミーティング、ファイル、チャットといった業務データへセキュアにアクセスし、膨大な量の情報を分析して、高度な専門知識をオンデマンドで提供する。

　「Researcher」と「Analyst」は新しい「Frontier」プログラムの一環として、4月から「Microsoft 365 Copilot」ライセンスユーザーに提供される。

## Researcher

　「Researcher」は、複雑なマルチステップのリサーチを効率的に行うために設計されたエージェント。OpenAIの「Deep Research」モデルを基盤としており、「Microsoft Graph」やWeb検索を統合しながら、ユーザーのデータをリアルタイムで分析する。

[![](https://asset.watch.impress.co.jp/img/wf/docs/2002/484/image2_l.png)](https://forest.watch.impress.co.jp/img/wf/docs/2002/484/html/image2.png.html) 専門家の代わりに徹底的な調査を行ってくれる「Researcher」

　生成された回答は専門家のサーベイに匹敵する質をもち、キャンバスアプリ「Pages」でチームと共有しながら編集が可能だ。必要な専門知識がすぐさま得られるため、日常的な業務が大いに効率化されるだろう。

![](https://www.youtube.com/watch?v=lfruwkpqvk4)Researcher: A reasoning agent in Microsoft 365 Copilot

## Analyst

　「Analyst」は、複雑なデータセットを扱えるデータサイエンティストのようなエージェントだ。OpenAIの「o3-mini」推論モデルをベースとしており、「Python」コードを生成・実行したり、データセットのコンテキストやビジュアライゼーションを提供する。思考過程や生成された「Python」コードをリアルタイムで確認し、その信頼性を検証することも可能。

[![](https://asset.watch.impress.co.jp/img/wf/docs/2002/484/image3_l.png)](https://forest.watch.impress.co.jp/img/wf/docs/2002/484/html/image3.png.html) 「Python」で大量かつ複雑な業務データを分析してくれる「Analyst」

　社員に大規模データを扱う「Python」スキルがなくても、新製品の需要予測や顧客の購買パターンの視覚化、収益予測といった分析が行えるというわけだ。

![](https://www.youtube.com/watch?v=9O3CoP8rEkY)Analyst: A reasoning agent in Microsoft 365 Copilot

　そのほかにも、「Microsoft Copilot Studio」には「Deep reasoning」と「Agent flows」が同日より導入される。「Deep reasoning」は応答により長い時間を要するが、より複雑なタスクをより正確に完了できるのが魅力。「Agent flows」は、エージェントの動作を分割・構造化し、複数を組み合わせることで条件、例外、ループなどの複雑な意思決定を手軽に設計できる機能だ。自然言語の指示またはグラフィカルユーザーインターフェイスを使用して作成できるため、誰もが「Copilot Studio」で簡単に構築できる。
